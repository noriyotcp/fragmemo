**Snippet manager built with Electron, Lit & monaco-editor**

At this time, this supports for macOS only.

## Development

```sh
yarn install
yarn electron:dev

# build the app (depends on your architecture)
yarn app:build
# for Mac x64
yarn app:build:mac:x64
# for Mac arm64
yarn app:build:mac:arm64
```
